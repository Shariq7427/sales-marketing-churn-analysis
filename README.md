# Sales & Marketing Customer Churn Prediction

## Project Overview

This project analyzes customer behavior and predicts customer churn using machine learning techniques.

The project includes:
- Data cleaning
- Exploratory Data Analysis (EDA)
- Data visualization
- Machine Learning modeling
- Model evaluation

---

## Dataset

Dataset Source:

https://www.kaggle.com/datasets/bhaskerpaul/sales-and-marketing-dataset/data

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow

1. Data Collection
2. Data Cleaning

6. Model Evaluation
